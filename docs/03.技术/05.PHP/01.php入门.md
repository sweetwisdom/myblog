# php 入门

## 什么是 php

> PHP（“PHP: Hypertext Preprocessor”，超文本预处理器的字母缩写）是一种被广泛应用的开放源代码的多用途脚本语言，它可嵌入到 HTML 中，尤其适合 web 开发

## 特征:

- PHP 是一门语言，服务器端的脚本语言
- 适合开发 WEB，实现功能
